
## Laravel Platform


## Install

### For development
```$xslt
composer install
npm install
npm run dev ?? npm run prod
php artisan migrate --seed
```

### For testing
```
composer install
npm install
npm run watch
php artisan migrate:fresh --seed
php artisan serve
```

## Auth

- Admin
```
- U: admin@admin.com

```

- User

